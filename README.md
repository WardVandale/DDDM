# DDDM
Dungeons &amp; Dragons - Dungeon Master

## Requirements

1. Python
2. Pip packages
    1.  Flask
    2.  Json

## Install Pip Packages:

```bash
pip install flask
pip install json
```

## Run the app:

run the command

```bash
python app.py
```
